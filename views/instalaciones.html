Instalaciones

Creamos el proyecto

Podemos usar cualquier herramienta, preferiblemente laragon, pero eso no significa que no hay otros medios para crear proyectos.


A continuacion el codigo


composer global remove laravel/installer
composer global require laravel/installer
laravel new nombre-del-proyecto

Instalación vue3

Vue3 es una herramienta muy utilizada para establacerla como herramienta forntend del proyecto, también se encuentra la instalacion del vue router.

A continuacion el codigo


cd nombre-del-proyecto
npm install vue
npm install vue-loader
npm install @vitejs/plugin-vue
npm install vue-router@4
code .

Configuración Vue3

Vue3 tiene la siguiente configuracion en el archivo:

vite.config.js

Importar codigo


import vue from '@vitejs/plugin-vue';

Importar configuración


export default defineConfig({plugins: [vue(),]});
					


vue(),

Tener cuidado debido a que aquí solo se le agrega una linea de codigo al archivo, se recomienda solo copiar el segundo codigo.

Instalar la api


php artisan install:api

Seguir la siguiente estructura

resources
│
├── css
│   ├── app.css
│   └── vue.css
│
├── js
│   ├── app.js
│   ├── bootstrap.js
│   └── vue
│       ├── App.vue
│       ├── main.js
│       ├── routes.js
│       └── components
│           └── EjemploComponent.vue

Comandos para la estructura

						
cd resources
mkdir css
echo. > css\vue.css
mkdir js
mkdir js\vue
echo. > js\vue\App.vue
echo. > js\vue\main.js
echo. > js\vue\router.js
mkdir js\vue\components
echo. > js\vue\components\listas.vue
echo. > js\vue\components\registros.vue
mkdir js\vue\jsComponents
cd ..
echo. > resources\views\vue.blade.php

Comandos para la BD


mysql -u root -p
					

CREATE DATABASE nombre_de_la_base_de_datos;
					

exit

npm install vue-router@next

Instalar Taildwind

No es necesario, pero de todos modos aquí está la instalación

Instalacion

						
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm install @headlessui/vue @heroicons/vue

Así se instala tailwind junto con sus dependencias y se expone.

Configuraciones

En el archivo tailwind.config.js agregamos en el array de content


export default{
	content:[
		"./resources/js/**/*.{vue,js,ts,jxs,tsx}",
	]
}

importacion en css

creamos un archivo css de vue con lo siguiente


@import 'tailwindcss/base';
@import 'tailwindcss/components';
@import 'tailwindcss/utilities';

y lo importamos en el main.js por lo general es de la siguiente manera import '../../css/vue.css';
